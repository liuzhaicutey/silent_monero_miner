# Silent Monero Miner
This is a C++ and Python-based framework for the automated deployment of the XMRig miner. 
# Usage
- For C plus2 Exe/DLL: Compile the source using Visual Studio and run via CMD: silent_miner.exe <Wallet_Address> (EXE) or hardcode the address and use injectors (DLL).
- For Python: Use CMD and run: python silent_miner.py
